##### 社交创伤维度（题目13-16）
- **题目13**: 朋友反目成仇背叛

- **题目25**: 表白被嘲弄拒绝感情恐惧
- **题目26**: 重要场合出丑长期羞耻
- **题目27**: 被PUA对自己价值产生怀疑
- **题目28**: 重大过失持续自责（车祸/连累他人等）

#### 2. 统一答案格式
**所有28题首选项统一为"从未经历过"**
- 提升用户体验一致性
- 减少选择困惑  
- 符合心理测评标准
- 明确的"无经历"界定

#### 3. 雷达图计算修正
**修复了严重的数据显示错误**：
- ❌ **原计算错误**: 除以3再乘以100
- ✅ **修正为**: 除以12再乘以100  
- **修复效果**: 雷达图现在显示完全准确的百分比
