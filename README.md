# 心结解开评估系统 - 优化版本

## 优化说明

本版本对所有28道题目的选项进行了优化，使语言更加贴近日常生活，提升用户体验。

### 主要优化内容

#### 1. 语言口语化
- **优化前**："偶尔有矛盾"、"轻微影响"、"明显影响"
- **优化后**："偶尔会有小矛盾"、"有一点影响"、"影响比较明显"

#### 2. 表达更完整
- **优化前**："没有恐惧"、"认知正常"
- **优化后**："没有恐惧感"、"认知很正常"

#### 3. 减少理解障碍
- **优化前**："深刻创伤"、"严重扭曲"
- **优化后**："留下深刻创伤"、"严重扭曲认知"

#### 4. 统一表达风格
- **优化前**："略有顾虑"、"明显犹豫"
- **优化后**："有点担心"、"比较犹豫"

### 具体优化示例

#### 亲密关系破裂维度
- **题目1**: "有过轻微经历" → "偶尔有类似情况"
- **题目2**: "能够快速释怀" → "很快能走出来"  
- **题目3**: "仍保持信任" → "仍然愿意相信"

#### 亲子关系冲突维度
- **题目5**: "关系和谐" → "关系很好"
- **题目6**: "没有创伤" → "没有不良影响"
- **题目7**: "没有问题" → "没有这方面的担心"

#### 童年阴影创伤维度  
- **题目21**: "偶尔缺乏" → "偶尔缺乏安全感"
- **题目22**: "关系正常" → "关系很正常"
- **题目23**: "没有恐惧" → "没有恐惧感"
- **题目24**: "认知正常" → "认知很正常"

#### 公开受辱经历维度
- **题目26**: "略有紧张" → "有点紧张"
- **题目27**: "不会避免" → "不会刻意避免"

### 优化效果

1. **更容易理解** - 使用日常用语，减少专业术语
2. **减少填写疑虑** - 表达更具体明确
3. **提高完成率** - 降低理解门槛
4. **保持简洁性** - 维持4点量表的易用性

### 系统功能

- ✅ 7个心结维度评估
- ✅ 28道核心题目
- ✅ 可视化雷达图分析
- ✅ 详细结果报告
- ✅ 数据本地保存（隐私保护）
- ✅ 响应式设计（支持手机/平板）

### 部署方式

#### 方式一：直接部署
1. 将 `index.html` 上传到任何静态网站托管服务
2. 支持 GitHub Pages、Netlify、Vercel 等

#### 方式二：本地预览
```bash
# 使用任意HTTP服务器
python -m http.server 8000
# 或
npx serve .
```
